# Medical Record Validator

A Python program that validates medical record data using custom validation rules.

## Description

This project checks whether patient records follow the expected format and data constraints. It validates fields such as patient ID, age, gender, diagnosis, medications, and visit ID, and reports any invalid records.

This project was completed as part of the **FreeCodeCamp Python Certification**.

## Features

- Validates patient record data
- Checks data types and required fields
- Uses regular expressions for ID validation
- Detects invalid or missing values
- Provides clear error messages

## Technologies Used

- Python 3
- Regular Expressions (`re` module)
